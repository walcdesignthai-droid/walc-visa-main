## 1. タイ入国の法的基本

### 1-1. 180日ルール(税法基準)

```
┌────────────────────────────────────────────────┐
│  タイ税法の基本                                  │
├────────────────────────────────────────────────┤
│                                                 │
│  年間180日以上滞在する外国人                     │
│              ↓                                  │
│  「居住者」とみなされる                          │
│              ↓                                  │
│  タイでの納税義務が発生                          │
│                                                 │
└────────────────────────────────────────────────┘
```

→ **この180日ルールが、タイ・イミグレの判断基準のベース**になっている。

### 1-2. ノービザ(Visa Exemption)の現行ルール

| 項目 | 内容 |
|---|---|
| 期間 | **60日**(2026年5月時点・変更情報はフェイクニュース) |
| 延長 | タイ国内イミグレで30日延長申請可 |
| **最大滞在** | **合計90日**(60+30) |
| 対象 | 日本含むVISA免除国 |
| 公式リスト | https://fukuoka.thaiembassy.org/jp/page/visa20724 |

#### ⚠️ 30日に戻る話題について

「ノービザが30日に戻る」という情報が頻繁に出回るが、**2026年5月時点では60日のまま**。
公式発表があるまで現状を維持。フェイクニュース・ハヤトチリ情報に注意。

## 2. イミグレ担当官の裁量

### 2-1. 担当官に与えられた権限

```
┌──────────────────────────────────────────────┐
│  タイ・イミグレ担当官の判断基準                │
├──────────────────────────────────────────────┤
│                                               │
│  ★ 明確な基準は存在しない                     │
│  ★ 「怪しい」と思われるだけで入国拒否可能     │
│  ★ アメリカ等の入国管理と類似                 │
│                                               │
└──────────────────────────────────────────────┘
```

### 2-2. 担当官がチェックする主要ポイント

最大の判断基準:

> **「入国者の年間タイ滞在日数が180日を超える見込みがあるか?」**

これが**最重要**。180日を超える見込みのある人は厳しく審査される。

### 2-3. パスポートスキャンで見える情報

```
[パスポートスキャン時に表示される情報]
   ├─ 過去の入国履歴
   ├─ オーバーステイ歴
   ├─ アラートの有無
   └─ 過去のトラブル記録
```

→ 担当官は即座に過去履歴を確認可能。

## 3. アラート制度(重要)

### 3-1. アラートの種類

| 種類 | 内容 | 期限 |
|---|---|---|
| **通常アラート** | トラブル・拒否歴等 | **毎年1月1日にリセット** |
| **永久アラート** | 悪質な履歴 | **消えない**(継続記録) |

### 3-2. アラートが付くトリガー

| 行為 | アラートの可能性 |
|---|---|
| 一度の入国拒否 | 通常アラート |
| オーバーステイ | 通常〜永久 |
| イミグレでの暴言・トラブル | 通常〜永久 |
| 何度もビザラン | 通常 |
| 偽造書類提出 | 永久 |
| 違法就労摘発 | 永久 |

### 3-3. アラートなしでも厳しいチェックを受けるケース

アラートがなくても、**「怪しい」と判断**されると:

- パスポートのスタンプページを1ページずつ確認
- 滞在目的・宿泊先・所持金等の質問攻め
- 別室送りの可能性

## 4. 入国拒否される典型ケース

### 4-1. 高リスク・典型パターン

```
┌──────────────────────────────────────────────┐
│  ★ 入国拒否される可能性が高いケース           │
├──────────────────────────────────────────────┤
│                                               │
│  VISAランを2回以上行っており、                 │
│  トータル滞在日数が100日前後に達している人     │
│                                               │
│  → 次の入国で180日を超える可能性があるため     │
│  → 入国拒否のリスクが高まる                    │
│                                               │
└──────────────────────────────────────────────┘
```

### 4-2. リスクレベル別チェック(ノービザ運用継続中の顧客向け)

> 💡 **本表は、まだDTV等のVISAを保有していない顧客の入国リスク評価です。**
> WALCの基本方針は**全顧客にDTV取得を推奨**することであり、本表は
> 「DTV取得前の直近の入国」「DTV以外のVISAで一時的に入国する場合」のリスク評価です。

| リスクレベル | 該当ケース | WALC推奨対応 |
|---|---|---|
| **低** | 初回入国・滞在歴なし | 通常入国でOK + 中長期的にDTV検討 |
| **中** | ノービザ1回利用・トラブルなし | 通常入国OK + DTV提案開始 |
| **高** | ビザラン2回以上・年間100日超 | **早急にDTV取得・直近の入国のみ空港サポート併用検討** |
| **最高** | 入国拒否歴・オーバーステイ歴 | **DTV取得を最優先・直近の入国は空港サポート必須** |

### 4-3. WALCの基本方針(再確認)

```
[原則]
   全ての顧客に DTV VISA を第一推奨

[空港イミグレサポートの位置づけ]
   ・DTV取得者には基本的に不要
   ・以下の「万が一」のケースで利用
     - DTV取得が間に合わない直近の入国
     - 過去にアラート発生があり、直近の入国リスクが高い
     - 取得済みでも特別に並びたくない方の任意利用

[禁止事項]
   ✗ DTVと空港サポートを抱合せで「セット」として推奨しない
   ✗ DTV取得後に「念のため」と空港サポートを勧めない
```

## 5. 30日延長申請の実務(ノービザ→90日)

### 5-1. 申請場所・費用

| 項目 | 内容 |
|---|---|
| 申請場所 | タイ国内のイミグレーション(各県の入国管理事務所) |
| 費用 | **1,900 THB** |
| 申請タイミング | 滞在期限の前(余裕を持って) |
| 必要書類 | パスポート・申請書・写真・現在の住所証明 |

### 5-2. 申請の流れ

```
[Step 1] イミグレに来訪
       ↓
[Step 2] 申請書・写真・パスポート提出
       ↓
[Step 3] 1,900 THB支払
       ↓
[Step 4] 当日 or 後日にスタンプ発給
       ↓
[Step 5] +30日の延長完了
```

### 5-3. 延長申請が却下されるケース

| ケース | 対応 |
|---|---|
| 過去にビザラン多数 | **空港サポートで再入国の選択** |
| 何度も延長申請を繰り返し | **DTV取得 or 他VISA推奨** |
| イミグレでアラート | **WALC専門スタッフ相談** |

## 7. 入国時のトラブル対応

### 7-1. イミグレで止められた場合の鉄則

```
鉄則1: 暴言・大声を絶対に出さない
       (記録され、永久アラートのリスク)

鉄則2: 質問には簡潔・正直に答える
       (盛らない・嘘をつかない)

鉄則3: 「分かりません」も明確な答え
       (迷うのが最もリスク)

鉄則4: 別室送りでも冷静に
```

## 8. ノービザ滞在の最適運用

### 8-1. リスク別の運用戦略

| 滞在パターン | リスク | WALC推奨 |
|---|---|---|
| 年1〜2回・各30〜60日 | 低 | ノービザのまま運用 |
| 年3〜4回・累計90〜150日 | 中 | DTV検討開始 |
| 年5回以上・累計150日超 | 高 | **DTV取得を強く推奨** |
| 何度もビザラン・過去アラートあり | 最高 | **DTV取得を最優先・直近の入国のみ空港サポート併用検討** |

> 💡 注:DTV取得後は空港サポートは基本不要です。
> 空港サポートが必要となるのは「DTV取得が間に合わない直近の入国」「過去アラートのある直近の1〜2回の入国」のみ。

### 8-2. ビザラン疲れの典型シグナル

```
☐ 年に4回以上タイに行っている
☐ ビザラン手配が面倒に感じる
☐ イミグレでチェックが厳しくなった気がする
☐ 30日延長を渋られた事がある
☐ 年間滞在日数が180日に近づいている
```

→ **2つ以上該当**したら、DTV提案のタイミング。

## 9. 顧客への説明用キーメッセージ

### 9-1. ノービザ運用について

> 「ノービザは現在60日(2026年5月時点)許可されますが、**ビザラン規制が強化**されており、何度も繰り返すとイミグレで止められるリスクが高まります。**何かしらのVISA取得**を強くおすすめします。」

### 9-2. アラートについて

> 「過去にトラブルがあった方は、パスポートをスキャンした時点でアラートが表示されます。**毎年1月1日にリセット**されますが、**悪質な履歴は消えない**ケースもあるため、ご不安な方は事前に空港サポートをご利用ください。」

### 9-3. 180日ルールについて

> 「タイの法律上、**年間180日以上滞在すると居住者**とみなされます。これがイミグレの判断基準のベースなので、180日超を希望する方は**居住者向けVISA(DTV/NON-O/LTR/Privilege)**への切替を推奨します。」
