## 0. 本ドキュメントの位置づけ

**2026年4月の制度変更により、銀行口座開設ルールが大幅に変わりました。**
この変更は **複数事業マニュアルに横断的に影響**するため、共通リポジトリで一元管理します。

全マニュアルの該当箇所は、本ドキュメントを参照すること。

## 1. 制度変更の全体像

### 1-1. 旧制度 vs 新制度

```
┌─────────────────────────────────────────────────┐
│                                                 │
│  [旧制度 〜 2026年3月]                          │
│                                                 │
│  ✅ 通常の口座開設が可能                         │
│  △ DTV保有者も場合により口座開設可能            │
│                                                 │
└─────────────────────────────────────────────────┘
                       ↓
              2026年4月から大幅変更
                       ↓
┌─────────────────────────────────────────────────┐
│                                                 │
│  [新制度 2026年4月〜]                           │
│                                                 │
│  ✕ 通常の口座開設は不可                         │
│  ✕ DTVでは口座開設不可                          │
│                                                 │
│  ✅ 開設可能なVISA:                              │
│     ・NON-Immigrant VISA(NON-B/NON-O/NON-ED)  │
│     ・Thailand Privilege                       │
│     ・LTR VISA                                 │
│                                                 │
│  ⚠️ 口座を使えるようになるのはVISA取得後         │
│                                                 │
└─────────────────────────────────────────────────┘
```

## 2. VISA別 口座開設可否マトリクス

### 2-1. 全VISA口座開設可否表

| VISA種別 | 口座開設 | 備考 |
|---|---|---|
| **観光ノービザ** | ✕ 不可 | 短期滞在のため |
| **TR15(VOA)** | ✕ 不可 | 15日のみ |
| **TR60(観光VISA)** | ✕ 不可 | 観光のため |
| **TR180(マルチプル観光)** | ✕ 不可 | 観光のため |
| **DTV VISA** | **✕ 不可** | **2026年4月変更** |
| **NON-B(ビジネス)** | ✅ 可能 | WP取得連動 |
| **NON-O Retirement** | ✅ 可能 | リタイア定番 |
| **NON-O 結婚** | ✅ 可能 | — |
| **NON-O 家族** | ✅ 可能 | — |
| **NON-ED(学生)** | ✅ 可能 | — |
| **Thailand Privilege(全ティア)** | ✅ 可能 | サポート付き |
| **LTR VISA(全カテゴリ)** | ✅ 可能 | 最強カテゴリ |

### 2-2. 重要ポイント

```
★ 観光カテゴリのVISA(DTV含む)は全て口座開設不可
★ 居住者向けVISA(NON-Immigrant系)は開設可能
★ Thailand PrivilegeとLTRは観光カテゴリだが、
  制度設計上、銀行口座開設サポートが組み込まれている
```

## 5. 営業上の活用方法

### 5-0. 基本前提

```
★ 重要 ★

口座開設希望の顧客への対応は、
「DTV第一推奨」の例外的なケースとして扱う。

[顧客対応の判断軸]
   Q. 銀行口座開設は「必須」か「あれば嬉しい」か?
   
   ┌─ 「必須」(明確に必要)
   │       ↓
   │   DTV以外を提案(NON-O / Privilege / LTR)
   │
   └─ 「あれば嬉しい」(本当は不要)
           ↓
       DTV第一推奨を維持
       (現金主義の生活で十分なケースも多い)

→ 口座開設を理由に高額VISAを推奨することは
  顧客利益に反する場合がある。
  まず「本当に口座が必要か?」をヒアリングする。
```

### 5-1. クロスセル機会(口座が「必須」な顧客のみ)

```
[シナリオA] DTV保有者で口座が「必須」になった顧客
       │
       ▼
   Thailand Privilege Bronze追加(650,000 THB)
   or LTR切替(50,000 THB + 要件)
       
[シナリオB] 50歳以上で口座が「必須」
       │
       ▼
   NON-O Retirement(13,000〜72,000 THB)
   → 口座開設サポート +6,000 THB
       
[シナリオC] 富裕層で口座が「必須」
       │
       ▼
   LTR WGC or Thailand Privilege
   → 「口座開設込み」を訴求
```

### 5-2. 「DTV+口座」幻想を持つ顧客への対応

```
[よくある誤解]
   「DTV取れば口座も開けるんですよね?」

[標準回答]
   2026年4月の制度変更により、
   DTVでは銀行口座を開設できなくなりました。

   ただし、まずお伺いしたいのですが、
   タイで「銀行口座が必須」のご事情はございますか?

   ・タイで給与受取等の必要性 → 必須
   ・タイで賃貸物件の契約 → 場合により必須
   ・「あれば便利かな?」程度 → 多くの方は不要

   ご事情によっては、DTV単体で
   十分にタイ滞在できるケースも多くございます。

   もし口座開設が必須であれば、
   NON-O Retirement / Thailand Privilege / LTR Visa
   いずれかをご検討いただく形になります。
```

### 5-3. 口座希望顧客への提案順序

口座開設が「必須」と確認された顧客への提案は以下の順序:

```
[第1優先] 50歳以上 → NON-O Retirement(13,000〜72,000 THB)
   ※ DTV+α の予算感で最もコスパが良い

[第2優先] 富裕層・面倒回避 → Thailand Privilege
   ※ コンシェルジュ込み

[第3優先] 高所得 → LTR Visa
   ※ 専門スタッフ転送
```

#### リタイアメントビザの優位性訴求(口座希望ケースのみ)

```
[文脈] 50歳以上で口座開設が必須な顧客への提案

「50歳以上で残高80万THBがあれば、
わずか13,000 THBから(日本国内E-VISA)
口座開設可能なVISAが取得できます。

DTVをご検討中とのことですが、
お客様の場合は銀行口座が必須とのことなので、
DTV(45,000〜60,000 THB・口座開設不可)よりも
リタイアメントビザの方がお客様の目的に合っております。」
```

## 7. 顧客対応の決め台詞集

### 7-1. 「DTVだけで口座も開きたい」への対応

```
「申し訳ございませんが、
DTV単独では銀行口座開設ができなくなりました。

ただし、ご安心ください。
お客様のご状況に応じて、
DTV + Thailand Privilege Bronze(口座可)、
NON-O Retirement(50歳以上の場合)など
複数の選択肢がございます。

ご相談を進めさせてくださいませ。」
```

### 7-2. リタイア層への提案

```
「50歳以上で残高80万THB相当がございましたら、
NON-O リタイアメントが最もコスパの良い選択肢です。

たった13,000 THBから取得可能で、
口座開設サポート(+6,000 THB)も含めて
20,000 THBほどで完結します。

DTVや他VISAと比べて圧倒的に安く、
かつ銀行口座まで開けるのは
弊社の強みでもございます。」
```

### 7-3. 富裕層・面倒回避層への提案

```
「タイ移住を本格的にご検討でしたら、
Thailand Privilege か LTR Visaをおすすめします。

両方とも銀行口座開設が含まれており、
ビザの更新や90日レポートの煩雑さからも解放されます。

LTRは要件が高いですが税優遇が大きく、
Privilegeは要件不問で取得しやすいのが特徴です。

詳細を弊社の専門スタッフよりご案内させて頂きます。」
```
