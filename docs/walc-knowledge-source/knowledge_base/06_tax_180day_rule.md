## 1. タイ税法の居住者判定

### 1-1. 基本ルール

```
┌────────────────────────────────────────────────┐
│  タイ居住者判定の基準                            │
├────────────────────────────────────────────────┤
│                                                 │
│  暦年(1月1日〜12月31日)中に                   │
│  タイに 180日以上 滞在 = タイ税務上の居住者     │
│                                                 │
│  → タイ国内・海外問わず所得への課税対象         │
│  (条件あり)                                    │
│                                                 │
└────────────────────────────────────────────────┘
```

### 1-2. 重要な前提

- 計算は**暦年単位**(1月〜12月)
- 180日**未満**なら非居住者(タイ税務上)
- VISA種別と居住者判定は**別物**(DTVでも180日超なら居住者扱い)

---

## 2. 居住者の税扱い(2024年制度変更以降)

### 2-1. 2024年〜の遡及課税ルール

```
[旧制度(〜2023年)]
   タイで稼いだ所得のみ課税
   海外所得は「翌年以降に持ち込めば非課税」

[新制度(2024年〜)]
   タイ居住者は、海外所得を
   「同年中にタイに持ち込んだ場合に課税」
   ※ 国際的な税務動向への対応
```

### 2-2. 2025-2026年の運用強化

- **2024年制度の運用が強化**
- 日本含む二国間租税条約との関係で実務複雑化
- WALCでは**専門スタッフ・税理士への転送**を標準対応

## 4. DTVの税扱い(よくある誤解)

### 4-1. 「DTVは納税義務なし」は正確には...

```
[誤解されがちな説明]
   DTV = 非居住者向けVISA = 納税義務なし

[正確な説明]
   DTV はVISA種別としては「非居住者向け(観光カテゴリ)」
   しかし、実際の税法上の居住者判定は
   「年間180日以上滞在しているか」が基準
   → DTV保有者でも180日超滞在すれば「タイ税務居住者」扱い
```

### 4-2. 顧客への正しい説明

```
[LINE / 営業時の正確な説明]

DTVは非居住者向けVISA(観光カテゴリ)なので、
基本的に納税義務は発生しません。

ただし、年間180日以上タイに滞在する場合は、
タイ税法上は「居住者」とみなされる可能性があり、
その場合は別途税務相談が必要となります。

180日未満の運用なら問題ございません。
```

→ 「基本的に」を**必ず付与**することがコンプライアンス上重要。

### 4-3. 顧客向け運用アドバイス

| 滞在パターン | 税扱い |
|---|---|
| 年間180日未満 | 非居住者・タイ国外所得は非課税 |
| 年間180日以上 | 居住者扱い・税務相談推奨 |

→ DTVは**180日連続滞在 + ビザラン**で運用するのが最も税務的にシンプル。

## 5. 90日レポート制度

### 5-1. 制度の本質

```
[90日レポートとは]
   居住者向けVISA保有者が
   90日以上タイに滞在する場合に
   「現在タイにいます」と報告する制度

[本来の対象]
   NON-Immigrant VISA保有者(居住者)
```

### 5-2. VISA別の90日レポート扱い

| VISA | 90日レポート |
|---|---|
| NON-B | 必須・罰則あり |
| NON-O Retirement | 必須・罰則あり |
| NON-ED | 必須・罰則あり |
| **DTV** | **実質不要**(罰則なし) |
| **Thailand Privilege** | 代行サポートあり |
| **LTR** | **年次レポートに軽減**(90日不要) |
| 観光ビザ・ノービザ | 不要 |

### 5-3. DTVの90日レポート — 公式と実情の違い

```
[公式見解]
   タイのVISA制度上、90日以上の滞在になるので
   90日レポートは必要(全VISA共通)

[弊社見解(実情)]
   DTVは観光カテゴリのVISA
   90日レポートが必要(罰則規定がある)のは
   NON-Immigrant(居住者向け)VISAの場合

   実際にDTVでの90日レポートは罰則などなく、
   かつ出入国時に全てリセットとなります。

   なので、実質不要とご案内しております。
   ご不安なようでしたらレポートも可能ですし、
   やっていなくても何も変わらないのが実情です。

[弊社実績]
   DTV制度発足から2年経過、
   イミグレでレポート未提出によるペナルティや
   止められた報告は一切なし
```

### 5-4. LTRの大幅軽減

LTR Visa保有者は **90日レポート → 年次レポート** に軽減。
これがLTRの大きな魅力の一つ。

## 6. 日本との二国間租税条約

### 6-1. 日タイ租税条約の基本

```
[適用関係]
   日本居住者がタイで収入を得る場合
   タイ居住者が日本で収入を得る場合
   両国に居住者判定される場合

   → 二国間租税条約で重複課税を回避
```

### 6-2. 日本住民票との関係

| 状況 | 居住者判定 |
|---|---|
| 日本に住民票あり・タイ180日未満 | 日本居住者 |
| 日本に住民票あり・タイ180日以上 | 両国で判定→条約で調整 |
| 日本に住民票なし・タイ180日以上 | タイ居住者 |
| 日本に住民票なし・タイ180日未満 | どこの居住者でもない可能性 |

→ 複雑なケースは**税理士相談**を強く推奨。

## 7. 実務上の注意点

### 7-1. 「タイで稼いで日本に送金」のケース

```
[ケース] タイで仕事して報酬を得て、日本の口座に振込

[税務上の扱い]
   → タイ国内で発生した所得 = タイ課税対象
   → DTV保有者でも、タイで就労相当の行為があれば
     リスクあり

[WALC方針]
   → DTVは「タイ国内での就労不可」VISAなので、
     これに該当する顧客は要相談
```

### 7-2. 「日本で稼いだ所得をタイで使う」ケース

```
[ケース] 日本企業所属・日本給与をタイの生活費に充てる

[税務上の扱い]
   → 同年中にタイに送金 = タイ課税対象(2024年〜)
   → 翌年以降に送金 = 非課税(従来通り)

[WALC方針]
   → 居住者判定の有無で扱いが大きく変わる
   → 180日以下の滞在なら問題なし
   → 180日超の場合は税理士相談推奨
```

### 7-3. リモートワーカーの典型ケース

```
[ケース] 海外企業所属・タイでリモートワーク・年収80kUSD

[VISA選択肢]
   ① DTV ノマド(45,000 THB)
      → 180日以内運用なら税務問題なし
   ② LTR WFTP(50,000 THB)
      → 180日超滞在でも外国所得非課税

[判断軸]
   滞在期間と年収で最適VISAが変わる
```

## 9. 顧客像 → 税務観点での最適VISA

| 顧客像 | 滞在 | 推奨VISA | 税扱い |
|---|---|---|---|
| 短期滞在(180日未満) | ◯ | DTV / 観光 | 非課税 |
| 長期滞在・所得低い | ◯ | DTV | 課税対象だが累進低率 |
| 長期滞在・所得80kUSD超 | ◯ | **LTR WFTP/HSP** | 大幅税優遇 |
| 富裕層・配当中心 | ◯ | **LTR WGC** | 外国所得非課税 |
| リタイア・年金中心 | ◯ | **LTR WP** | 外国所得非課税 |
| タイで就労希望 | ◯ | NON-B / LTR HSP | LTRなら17% |
| 富裕層・面倒回避 | ◯ | Thailand Privilege | 通常課税 |
