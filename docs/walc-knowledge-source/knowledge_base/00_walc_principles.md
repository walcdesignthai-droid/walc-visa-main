# WALC最重要営業方針

> **このドキュメントはWALC VISA Consultingの最上位営業原則です。**
> 全マニュアル・全営業対応で最優先される。
> 他の知識ベース・仕様書はこの原則に従属する。

---

## 1. 最重要原則:DTV VISA 第一推奨

```
┌──────────────────────────────────────────────┐
│  WALCの第一推奨は、原則すべて DTV VISA         │
└──────────────────────────────────────────────┘
```

### 1-1. DTV第一推奨の理由

| # | 理由 | 数値・根拠 |
|---|---|---|
| 1 | **コストパフォーマンスが圧倒的** | エリート比で 1/10 以下 |
| 2 | **5年マルチプル・180日滞在** | 自由度が極めて高い |
| 3 | **取得率100%** | 2025年4月制度変更以降 |
| 4 | **実績豊富** | 累計200名以上の取得経験 |
| 5 | **最短7〜14日で取得** | E-VISA申請以降 |
| 6 | **タイにいながら申請完結** | 2025年8月福岡領事館オンライン化 |

### 1-2. DTV vs Thailand Privilege コスト比較

```
[DTV ソフトパワー(WALC料金)]
   60,000 THB / 5年
   = 12,000 THB/年
   = 約56,000円/年

[Thailand Privilege Bronze]
   650,000 THB / 5年
   = 130,000 THB/年
   = 約610,000円/年

→ DTVが Privilege の 約1/10 のコスト

[15年で換算]
   DTV取得3回(15年分): 約180,000 THB
   Privilege Diamond(15年): 2,500,000 THB
   → 約14倍の差
```

### 1-3. DTV提案の優先順位

```
[第一推奨] DTV VISA(全ての顧客にまず提案)
       ↓
   ├─ ソフトパワー(60,000 THB) ← 最推奨
   ├─ ノマド(45,000 THB)
   └─ フリーランス(48,000 THB)

[DTV不可・要件未達の場合のみ代替提案]
   ├─ 50歳以上+残高80万THB → NON-O Retirement
   ├─ 富裕層・面倒回避    → Thailand Privilege
   ├─ 高所得・税優遇優先  → LTR Visa
   └─ 短期滞在のみ       → 観光VISA/ノービザ運用
```

## 2. DTV以外を提案する明確な条件

以下に該当する場合**のみ**、DTV以外を提案する:

```
✗ 50歳以上+残高80万THB保有
   → NON-O Retirement(13,000〜72,000 THB)を提案

✗ タイ国内で銀行口座開設が「必須」
   → NON-O / Privilege / LTR を提案
   ※ DTVでは口座開設不可(2026年4月変更)
   ※ 「あれば嬉しい」レベルでは DTV第一推奨維持

✗ タイで就労が必要
   → NON-B / LTR HSP を専門スタッフ転送

✗ 年間180日超滞在 + 外国所得が大きい(80kUSD超)
   → LTR Visa を専門スタッフ転送
   ※ 税優遇メリットがDTVを上回る

✗ 富裕層で時間と労力をお金で買いたい
   → Thailand Privilege を提案

[上記に該当しない場合]
   → 全て DTV VISA を第一推奨
```

## 3. 禁止事項

### 3-1. 抱合せ販売の禁止

```
✗ DTV + 空港イミグレサポート の「セット提案」
✗ DTV取得者に「念のため」と空港サポートを勧める
✗ DTV + 銀行口座開設サポート の抱合せ
   (DTVでは口座開設不可・2026年4月変更)
✗ 推測で「便利だろう」とパッケージ化
```

### 3-2. 推測による営業提案の禁止

```
✗ Yosukeから明示されていない営業構造を作らない
✗ パッケージ・セット価格を確認なしに提示しない
✗ 「便利だろう」という推測でクロスセル構造を作らない
```

### 3-3. 過度な保証表現の禁止

```
✗ 「絶対通ります」「100%保証」等の断定表現
✓ 「現状取得率100%」+ ※ただし書き
```

#### ただし書きテンプレ

```
※ 2025年4月〜2026年5月時点の弊社実績です。
  将来の取得を保証するものではありません。
  タイのVISA制度は予告なく変更される場合があります。
```

## 4. 空港イミグレサポートの位置づけ

### 4-1. 基本方針

```
[原則]
   DTV取得者には空港サポートは「基本不要」

[利用シーン] 「万が一」のケースのみ
   ・DTV取得が間に合わない直近の入国
   ・過去にアラート発生があり、直近の入国リスクが高い
   ・取得済みでも特別に並びたくない方の任意利用

[料金]
   ・通常: 6,000 THB(スワンナプーム事前予約)
   ・DTV取得者向け割引: 4,000 THB
     ※「DTV取得が間に合わない」万が一の場合のみ適用
```

### 4-2. 空港サポートを単体で必要とする顧客

```
[該当者]
   ・まだDTV未取得・直近の入国に不安がある
   ・過去にイミグレでアラート発生
   ・ビザラン2回以上で滞在歴100日前後

[対応]
   → 空港サポートを単体提供
   → 同時にDTV取得を強く推奨(長期的解決策)
   → 「DTVと空港サポートのセット」ではなく
     「DTV取得を準備しつつ、直近の入国だけ空港サポート」
     という時系列の使い分け
```

## 5. 顧客への提案フロー

### 5-1. 標準ヒアリング項目

```
1. 年齢(50歳ライン)
2. 国籍・現住所
3. 残高(50万THB / 80万THB)
4. 収入レベル(80kUSD/年ライン)
5. 滞在目的・期間
6. タイ入国歴
7. 緊急度・希望スケジュール
8. 銀行口座必要性(「必須」or「あれば嬉しい」)
9. 就労希望の有無
```

### 5-2. 提案フロー図

```
[ヒアリング完了]
       ↓
[第1判定] DTVの要件を満たすか?
   - 残高50万THB相当を3ヶ月キープ可能?
   - 日本の住民票/免許/入国スタンプあり?
       ↓
   ┌────────────────────────┐
   │ Yes:                   │
   │  → DTVを第一推奨       │
   │  → ソフトパワー優先     │
   │     (要件最も緩い)      │
   ├────────────────────────┤
   │ No:                    │
   │  → 代替VISA検討        │
   │     §2の条件で判定       │
   └────────────────────────┘
       ↓
[第2判定] 銀行口座が「必須」か?
   - 「必須」 → DTV以外を提案
   - 「あれば嬉しい」 → DTV維持
       ↓
[第3判定] 50歳以上か?
   - 50歳以上+残高80万THBあり → NON-O推奨可
       ↓
[第4判定] 富裕層・面倒回避希望か?
   - 高額予算OK → Privilege提案
       ↓
[最終提案] お客様に最適なVISAをご案内
```

## 6. 「エリート検討中」顧客への切り替えトーク

```
タイランドエリート(Privilege)もご検討頂きありがとうございます。

正直に申し上げますと、現在弊社では
DTV VISAを第一推奨としております。

【理由】
DTVは 約60,000 THB で5年取得できますが、
エリート Bronze は 650,000 THB と約10倍以上のコストです。

DTVで対応可能な範囲であれば、
費用を大幅に抑えて同等の自由度を実現できます。

ただし、銀行口座開設が必須な場合や、
20年単位の超長期滞在をお考えの場合は、
エリートの方が適しているケースもございます。

まずはお客様のご状況をお伺いし、
最適なご提案をさせてくださいませ。
```

## 7. 「DTV+口座」幻想を持つ顧客への対応

```
[よくある誤解]
   「DTV取れば口座も開けるんですよね?」

[標準回答]
   2026年4月の制度変更により、
   DTVでは銀行口座を開設できなくなりました。

   ただし、まずお伺いしたいのですが、
   タイで「銀行口座が必須」のご事情はございますか?

   ・タイで給与受取等の必要性 → 必須
   ・タイで賃貸物件の契約 → 場合により必須
   ・「あれば便利かな?」程度 → 多くの方は不要

   ご事情によっては、DTV単体で
   十分にタイ滞在できるケースも多くございます。

   もし口座開設が必須であれば、
   NON-O Retirement / Thailand Privilege / LTR Visa
   いずれかをご検討いただく形になります。
```
