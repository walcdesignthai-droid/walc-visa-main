## 2. 罰金体系(2026年5月時点・変更なし)

### 2-1. 基本罰金表

| 超過日数 | 罰金(1日あたり) | 最大罰金 |
|---|---|---|
| 1日〜 | **500 THB/日** | **最大 20,000 THB** |

#### 重要ポイント

- **1日でも超えた時点で罰金発生**(帰国便の日でも対象)
- 罰金は**空港等での出国時に支払う**
- **支払わないと出国不可**
- 40日超で上限の20,000 THBに到達(以降は累積しない)

### 2-2. 罰金計算例

| 超過日数 | 罰金額 |
|---|---|
| 1日 | 500 THB |
| 5日 | 2,500 THB |
| 10日 | 5,000 THB |
| 20日 | 10,000 THB |
| 30日 | 15,000 THB |
| **40日以上** | **20,000 THB(上限)** |

## 3. 入国禁止措置(重度違反)

### 3-1. 重度違反の処分一覧

| オーバーステイ期間 | 自主出頭時 | 逮捕・強制送還時 |
|---|---|---|
| 90日未満 | **罰金のみ** | **1年間入国禁止** |
| 90日以上〜1年未満 | **1年間入国禁止** | **3年間入国禁止** |
| 1年以上〜3年未満 | **3年間入国禁止** | **5年間入国禁止** |
| 3年以上〜5年未満 | **5年間入国禁止** | **10年間入国禁止** |
| 5年以上 | **10年間入国禁止** | **永久入国禁止の可能性** |

### 3-2. 自主出頭 vs 逮捕の決定的な差

```
┌──────────────────────────────────────────────┐
│  ★ 重要な判断軸                                │
├──────────────────────────────────────────────┤
│                                               │
│  自主出頭(自ら空港で申告して出国)             │
│       VS                                      │
│  逮捕・強制送還                                │
│                                               │
│  → 自主出頭のほうがペナルティが圧倒的に軽い    │
│  → 逮捕の場合、記録が残り続ける                │
│                                               │
└──────────────────────────────────────────────┘
```

### 3-3. 顧客対応の鉄則

> **「気付いた瞬間に出国準備」を最優先。** 
> 滞在が長引くほどペナルティが重くなる。
> 隠れて滞在続行は**絶対に避ける**(逮捕リスク)。

## 4. オーバーステイの発覚場面

### 4-1. 主要な発覚場面

| 場面 | リスクレベル |
|---|---|
| 空港での出国審査時 | 中(罰金支払で出国可) |
| タイ国内・警察の路上検問 | 高(その場で身柄拘束の可能性) |
| ホテル・病院・警察でのパスポート提出時 | 中〜高 |
| 違法就労・事件等での職質 | 最高(逮捕直結) |

### 4-2. 日常生活でリスクが高まる場面

```
☐ コンドミニアム契約更新時
☐ 銀行手続き時(口座更新等)
☐ 病院受診時
☐ ホテルチェックイン時
☐ 警察の取り締まり(交通違反等)
☐ 何らかの目撃者・通報者からの情報提供
```

→ **タイ国内ではパスポート提示の機会が頻繁**にあるため、長期オーバーステイは現実的に困難。

## 5. よくあるオーバーステイの原因

### 5-1. 典型的な5パターン

```
パターン1: 日数計算ミス
   例: 30日滞在のつもりが、「31日目の出国」でアウト
   → 入国日を「Day 0」と勘違いするケースが多い
   → 正しくは「入国日が30日許可の場合、29日後の同時刻まで」

パターン2: ビザ延長の失念
   例: 延長申請を忘れていた
   → イミグレ申請が必要なことを知らない

パターン3: 延長申請が却下
   例: イミグレで「もう延長できない」と言われた
   → 突然却下されるケースあり

パターン4: イミグレ来訪の面倒
   例: 「ちょっとくらい大丈夫」で放置
   → 1日でも違反は違反!

パターン5: 病気・事故(不可抗力)
   例: 入院・事故で出国できなかった
   → 診断書があれば情状酌量の可能性
```

### 5-2. ビザラン関連のオーバーステイ

| ケース | リスク |
|---|---|
| ビザラン手配が間に合わない | 中(早期気付きなら対応可) |
| 出国便のキャンセル・遅延 | 中(航空会社証明で軽減可能性) |
| 隣国でVISA取得失敗→帰国不能 | 高(WALCサポート案件) |

## 6. オーバーステイ対処法

### 6-1. 緊急時の対応フロー

```
[オーバーステイ発覚]
       ↓
[Step 1] 即座にWALCに連絡
       ↓
[Step 2] 状況整理
   - 何日超過しているか
   - 現在のVISA種別
   - パスポート残存期間
   - 過去のオーバーステイ歴
       ↓
[Step 3] 対応方針決定
   ┌─────────────────────────────┐
   │ ケースA: 90日未満             │
   │  → 自主出国・罰金支払い       │
   │  → 再入国戦略提案             │
   ├─────────────────────────────┤
   │ ケースB: 90日以上             │
   │  → 入国禁止期間の確認         │
   │  → 中長期戦略立案             │
   ├─────────────────────────────┤
   │ ケースC: 病気・不可抗力       │
   │  → 診断書取得サポート         │
   │  → イミグレ交渉               │
   └─────────────────────────────┘
       ↓
[Step 4] 出国手続き支援
       ↓
[Step 5] 出国後の再入国戦略
   (DTV取得・他VISA等)
```

### 6-2. 自主的なイミグレ来訪のメリット

```
✓ 罰金交渉の余地あり
✓ 延長申請への切替の可能性
✓ 担当官による情状酌量
✓ 「自主出頭」扱いでペナルティ軽減
```

→ 重度になる前に、**自主的にイミグレに行く**ことを強く推奨。

### 6-3. 不可抗力の証明書類

| 状況 | 必要書類 |
|---|---|
| 入院 | 医師の診断書(英語 or タイ語) |
| 事故 | 警察報告書 + 医師診断書 |
| フライトキャンセル | 航空会社発行のキャンセル証明 |
| 災害・コロナ等 | 公的機関発行の証明 |

→ 提出により、罰金免除・軽減の可能性あり。

## 7. WALC オーバーステイ対応サポート

### 7-1. 提供サポート内容

```
✅ 出国までのステップ整理
✅ 空港イミグレでの対応方法・注意点説明
✅ 必要書類(弁解文・診断書等)の翻訳支援
✅ 今後の再入国戦略のアドバイス
   (DTV・エリート・LTR等の活用)
✅ 緊急時の空港対応(空港サポート)
```

### 7-2. サポート料金(参考)

| サービス | 料金(目安) |
|---|---|
| 相談・戦略立案 | スポット見積 |
| 翻訳支援 | スポット見積 |

| 再入国時空港サポート | 6,000〜12,000 THB |
| 再入国戦略+VISA取得 | 各VISA料金 + 戦略フィー |

## 8. 入国禁止期間中の選択肢

### 8-1. 期間中にできること

| 状況 | 対応策 |
|---|---|
| 短期(1年) | 期間明けまで他国滞在・DTV準備 |
| 中期(3-5年) | 他国移住検討・タイ事業整理 |
| 長期(10年以上) | 永久的な他国拠点化 |

### 8-2. 期間明け後の戦略

```
[入国禁止期間明け]
       ↓
[第1選択肢] DTV VISA取得
   → 観光カテゴリ・アラート緩和の可能性
       ↓
[第2選択肢] Thailand Privilege
   → 富裕層向け・別途審査だが取得可能性あり
       ↓
[第3選択肢] LTR Visa
   → 要件満たせば最強選択
```

→ **オーバーステイ歴があってもVISA取得不可ではない**。期間明け後の戦略立案が重要。

## 9. 顧客向けキーメッセージ

### 9-1. オーバーステイ発覚時の初手メッセージ

```
ご相談ありがとうございます。
状況、承知いたしました。

オーバーステイ対応は、
1日でも早く動くことが何より大切です。

弊社にて以下をサポートさせて頂きます:
✓ 出国までのステップ整理
✓ 空港イミグレでの対応サポート
✓ 翻訳・書類準備
✓ 出国後の再戦略

まずは以下の情報をお送りくださいませ:
1. 現在のパスポート画像
2. 直近の入国スタンプ画像
3. 何日超過しているかの認識

すぐに対応方針をご案内いたします。
```

### 9-2. 「ちょっとくらい大丈夫」と思っている顧客への警告

```
「少しくらい大丈夫」は通用しません。
1日でも違反は違反です。

罰金:1日500 THB(最大20,000 THB)
さらに、今後の入国・VISA申請に大きく影響します。

早めに専門家(WALC)へご相談くださいませ。
```
