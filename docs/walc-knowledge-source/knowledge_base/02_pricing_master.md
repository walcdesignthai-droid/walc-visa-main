## 4. 価格改定時の運用ルール

### 4-1. 価格改定時のチェックリスト

```
□ 本ドキュメントの該当価格を更新
□ マスター仕様書 §4-B-2 の該当箇所を更新
□ 影響を受ける全マニュアルのCh.3「料金・支払い」を更新
□ LP・公式WEBサイトの料金表を更新
□ LINE自動応答メッセージの料金表を更新
□ Meta広告の料金訴求を確認・更新
□ 価格変更告知をLINE登録者に配信(必要に応じて)
```

### 4-2. 価格改定の影響範囲

```
[価格変更があった事業]
       ↓
[影響範囲]
   ├─ 該当事業マニュアル(Ch.3 料金)
   ├─ 該当事業マニュアル(Ch.13 返金規定)
   ├─ 価格マスター(本ドキュメント)
   ├─ マスター仕様書 §4-B-2
   ├─ 顧客動線マスターガイド(価格言及箇所)
   ├─ LP・WEBサイト
   ├─ LINE自動応答
   └─ 営業トーク台本
```

### 4-3. 為替変動への対応

```
[基本方針]
   料金はタイバーツ(THB)が正式表示
   日本円換算は参考値(申請日のレート反映)

[顧客への説明]
   「料金はすべてタイバーツ(THB)表示・税込です。
    為替レートにより日本円換算額は変動いたします。」

[ただし書きテンプレ]
   ※ 申請時の為替レートが反映されます
   ※ 円安・円高により円換算額が変動します
```

### 5-2. DTV提案の優先順位

```
[第一推奨] DTV VISA(全ての顧客にまず提案)
       ↓
   ├─ ソフトパワー(60,000 THB)←最推奨
   ├─ ノマド(45,000 THB)
   └─ フリーランス(48,000 THB)
       ↓
[DTV不可・要件未達の場合のみ代替提案]
       ↓
   ├─ 50歳以上・残高80万THB → NON-O Retirement
   ├─ 富裕層・面倒回避 → Thailand Privilege
   ├─ 高所得・税優遇優先 → LTR Visa
   └─ 短期滞在のみ → 観光VISA/ノービザ運用
```

### 5-3. DTV vs Thailand Privilege のコスト比較

```
[DTV ソフトパワー(WALC料金)]
   60,000 THB / 5年
   = 12,000 THB/年
   = 約56,000円/年

[Thailand Privilege Bronze]
   650,000 THB / 5年
   = 130,000 THB/年
   = 約610,000円/年

→ DTVが Privilege の 約1/10 のコスト

[15年で換算]
   DTV取得3回(15年分): 約180,000 THB
   Privilege Diamond(15年分): 2,500,000 THB
   
→ 約14倍の差
```

### 5-4. DTVを「全ての人にまず提案」する根拠

| 要素 | DTV | エリート | LTR |
|---|---|---|---|
| 価格 | ★★★ 最安 | ✗ 高額 | △ 政府費用安いが要件高 |
| 取得難易度 | ★★ 易しい | ★ 簡単 | ✗ 高所得・資産要件 |
| 滞在自由度 | ★★★ 5年マルチ180日 | ★★★ 5-20年 | ★★★ 10年 |
| 取得期間 | ★★★ 最短7-14日 | △ 1-3ヶ月 | ✗ 30-45日+書類準備 |
| 弊社実績 | ★★★ 200名以上・100% | △ 取次実績 | △ 専門スタッフ対応 |
| 顧客満足度 | ★★★ 高 | ★★ 高 | ★★ 高 |

→ **コスト・難易度・期間・実績、全ての軸でDTVが優位**。

### 5-5. 価格訴求の標準テンプレ

#### A. 「タイ移住のVISA、どれがいい?」への標準回答

```
タイ移住のVISAでしたら、まずDTV VISAを強くおすすめします。

【DTV VISAの強み】
✓ 5年マルチプル・1回180日滞在(延長で最大360日)
✓ コストはわずか 45,000〜60,000 THB
✓ 申請は完全オンライン・最短7〜14日で取得
✓ タイ国内からの申請にも対応(条件次第・状況により日本帰国が必要なケースあり)
✓ 弊社取得率100%(2025年4月以降)

タイランドエリートと比べると約1/10のコストで、
ほぼ同等以上の自由度を実現できます。

まずDTVが取得可能か、無料相談で確認させてくださいませ。
```

#### B. 「エリートを考えているのですが」への切り替えトーク

```
タイランドエリート(Privilege)もご検討頂きありがとうございます。

正直に申し上げますと、現在弊社では
DTV VISAを第一推奨としております。

【理由】
DTVは 約60,000 THB で5年取得できますが、
エリート Bronze は 650,000 THB と約10倍以上のコストです。

DTVで対応可能な範囲であれば、
費用を大幅に抑えて同等の自由度を実現できます。

ただし、銀行口座開設が必須な場合や、
20年単位の超長期滞在をお考えの場合は、
エリートの方が適しているケースもございます。

まずはお客様のご状況をお伺いし、
最適なご提案をさせてくださいませ。
```

#### C. 「DTV以外を考えるべきケース」

DTV以外を提案するのは、以下の明確な条件に該当する場合のみ:

```
[DTV以外を提案する明確な条件]

✗ 50歳以上+残高80万THB保有
   → NON-O Retirement(13,000〜72,000 THB)を提案

✗ タイ国内で銀行口座開設が必須
   → NON-O / Privilege / LTR を提案
   ※ DTVでは口座開設不可(2026年4月変更)

✗ タイで就労が必要
   → NON-B / LTR HSP を専門スタッフ転送

✗ 年間180日超滞在 + 外国所得が大きい(80kUSD超)
   → LTR Visa を専門スタッフ転送
   ※ 税優遇メリットがDTVを上回る

✗ 富裕層で時間と労力をお金で買いたい
   → Thailand Privilege を提案

[上記に該当しない場合]
   → 全て DTV VISA を第一推奨
```

## 6. 価格訴求時のコンプライアンス

### 6-1. 「コスパ1/10」表現の正確な使い方

```
[OK] DTV(60,000 THB)はエリート Bronze(650,000 THB)の
     約1/10以下のコストで、5年間タイに滞在できます。
     ※ 数値は2026年5月時点

[NG] エリートは無駄な出費
     → 他社サービスの否定はNG

[NG] DTVなら絶対通ります
     → 「絶対」表現はNG・「現状取得率100%」+ ただし書き
```

### 6-2. 「絶対」「100%」表現のただし書きテンプレ

```
※ 2025年4月〜2026年5月時点の弊社実績です。
  将来の取得を保証するものではありません。
  タイのVISA制度は予告なく変更される場合があります。
```

→ コスパ比較・取得率訴求時は必ず併記。
