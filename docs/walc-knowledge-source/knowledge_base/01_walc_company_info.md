```

       ↓
   ├─ ソフトパワー(60,000 THB)←最推奨
   ├─ ノマド(45,000 THB)

   ├─ 高所得・税優遇優先   → LTR Visa
   └─ 短期滞在のみ        → 観光VISA/ノービザ運用
```

### 抱合せ販売の禁止

```
✗ DTV + 空港イミグレサポート の「セット提案」は禁止
✗ DTV取得者に「念のため」と空港サポートを勧めない
✗ DTV + 銀行口座開設サポートの抱合せ提案は不可
   (DTVでは口座開設不可・2026年4月変更)

→ 単一サービスを推測でパッケージ化しない
→ 顧客の状況に応じた個別最適化を最優先
```

```
[基本] DTV取得者には空港サポートは不要

[利用シーン] 「万が一」のケースのみ
   ・DTV取得が間に合わない直近の入国
   ・過去にアラート発生があり、直近の入国リスクが高い場合
   ・取得済みでも特別に並びたくない方の任意利用

[料金体系]
   ・通常: 6,000 THB(スワンナプーム事前予約)
   ・DTV取得者向け割引: 4,000 THB
     ※「DTV取得が間に合わない」万が一の場合のみ適用
```

→ **このルールは全マニュアル・全営業対応の最上位原則**。
