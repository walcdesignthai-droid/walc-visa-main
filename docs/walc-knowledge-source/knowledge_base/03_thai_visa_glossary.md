## 2. NON-Immigrant VISA(居住者向け)

### 2-1. 基本特性

| 項目 | 内容 |
|---|---|
| 別名 | NONビザ(NON Immigrant)・居住者向けVISA |
| カテゴリ | 居住者(Resident) |
| 90日レポート | **必須・罰則あり** |
| 銀行口座開設 | **可能** |
| 納税義務 | あり(183日以上滞在で居住者判定) |

### 2-2. NON-B(ビジネスビザ)

| 項目 | 内容 |
|---|---|
| 用途 | ビジネス・就労 |
| サブカテゴリ | 複数(実務上は把握不要) |
| 取得難易度 | 高 |
| 就労 | **Work Permit(WP)が別途必要** |
| 期間 | 90日 → 1年延長可 |
| WALC対応 | **専門スタッフ転送** |

### 2-3. NON-O(Other)

NONビザの中で**最も多様**なカテゴリ。WALCの主力商品の一つ。

#### NON-O リタイアメント

| 項目 | 内容 |
|---|---|
| 対象 | **50歳以上** |
| 必須要件 | タイ口座+残高**80万THB以上** |
| 期間 | 1年 |
| 更新 | 1年毎(年次更新) |
| 就労 | 不可 |
| 90日レポート | 必須 |

→ WALCで料金表(13,000〜72,000THB)で対応。

#### NON-O 結婚VISA

| 項目 | 内容 |
|---|---|
| 対象 | タイ人配偶者を持つ外国人 |
| 必須要件 | 結婚証明書・配偶者の身分証等 |
| 期間 | 1年 |
| 就労 | 別途WP取得で可能 |
| WALC対応 | 案件により対応 |

#### NON-O 家族VISA(NON-B付帯)

| 項目 | 内容 |
|---|---|
| 対象 | NON-B保持者の配偶者・子(駐妻・駐在員家族) |
| 必須要件 | 主たる申請者のNON-B等 |
| WALC対応 | 案件により対応 |

#### NON-O ガーディアン(子育て)

| 項目 | 内容 |
|---|---|
| 対象 | タイ国内の子の保護者 |
| 必須要件 | 子の在学証明等 |
| WALC対応 | 案件により対応 |

#### NON-O ボランティア

| 項目 | 内容 |
|---|---|
| 対象 | NGO等のボランティア活動者 |
| 必須要件 | 招聘団体の証明書類 |
| WALC対応 | 案件により対応 |

### 2-4. NON-ED(教育・学生ビザ)

| 項目 | 内容 |
|---|---|
| 対象 | 語学学校・ムエタイ学校・大学・その他教育機関 |
| 期間 | 学校による(通常6ヶ月〜1年) |
| 90日レポート | 必須 |
| **WALC方針** | **現在非推奨**(DTVソフトパワーで代替可能なため) |

#### NON-ED が非推奨な理由

- DTVソフトパワーカテゴリでムエタイ等を5年マルチプル取得可能
- NON-EDは毎年更新・滞在制限あり
- DTVのほうが圧倒的に費用対効果高い

→ NON-EDを希望する顧客には**DTVソフトパワーへの誘導**を標準対応とする。

## 3. 観光カテゴリ(TR / Exemption)

### 3-1. 基本特性

| 項目 | 内容 |
|---|---|
| カテゴリ | 観光(Tourist) |
| 90日レポート | 不要(罰則なし) |
| 銀行口座開設 | **不可** |
| 納税義務 | 基本なし(183日超で居住者判定の可能性) |

### 3-2. TR15(VOA / Visa on Arrival)

| 項目 | 内容 |
|---|---|
| 別名 | VOA |
| 対象 | VISAが必須の31カ国(信用度の低い国) |
| **日本** | **対象外**(日本はVISA免除国) |
| 期間 | 15日 |
| 費用 | 2,000 THB |
| 申請場所 | タイ入国時の空港 |

### 3-3. TR60(60日観光VISA)

| 項目 | 内容 |
|---|---|
| 期間 | 60日 |
| 費用 | 2,000 THB |
| 申請場所 | タイ国外のタイ大使館・領事館 |
| **日本人にとって** | **メリット少**(現在ノービザ60日なので不要) |

### 3-4. Visa Exemption(ビザ免除 / 通称「w30」「w60」)

| 項目 | 内容 |
|---|---|
| 通称 | w30(ผ.30)、w60(ผ.60) |
| 名称由来 | タイ語の「ผ」が「w」に見えるため業界でそう呼ぶ |
| **現在(2026/5)** | **60日許可**(変更情報あるがフェイクニュース・現在60日のまま) |
| 30日延長 | タイ国内イミグレで申請可 → **最大90日** |
| 費用 | 無料(延長は1,900THB) |
| 対象国 | VISA免除国(日本含む) |

#### 公式リスト

VISA免除国の最新リスト:
https://fukuoka.thaiembassy.org/jp/page/visa20724

#### ⚠️ 重要 — フェイクニュース注意

「ノービザが30日に戻る」「VISA免除廃止」等のニュースが頻繁に出るが、
**2026年5月時点では60日のまま**。変更があるまで現状を維持。

### 3-5. TR180(マルチプル観光VISA)

| 項目 | 内容 |
|---|---|
| 期間 | 180日(マルチプル) |
| 費用 | 5,500 THB(26,000円) |
| 申請場所 | タイ国外のタイ大使館 |
| **位置づけ** | **DTV取得不可時の延命策**として利用されることが多い |

## 4. DTV VISA(Destination Thailand Visa)

→ **詳細はDTV専用マニュアルを参照**

要点:

| 項目 | 内容 |
|---|---|
| 制度開始 | 2024年7月 |
| カテゴリ | **観光VISA**(非居住者向け) |
| 期間 | 5年マルチプル / 180日滞在 / 180日延長で最大360日連続 |
| 残高要件 | 50万THB相当を**3ヶ月キープ**(2025年4月制度厳格化) |
| 90日レポート | **実質不要**(罰則なし・出入国でリセット) |
| 銀行口座開設 | **不可**(2026年4月制度変更後も継続不可) |
| 就労 | タイ国内での就労不可・海外企業のリモートワークOK |

WALC料金:
- ソフトパワー: 60,000 THB
- ノマド: 45,000 THB
- フリーランス: 48,000 THB

## 5. LTR Visa(Long-Term Resident)

### 5-1. 制度概要

| 項目 | 内容 |
|---|---|
| 制度開始 | 2022年9月 |
| 運営 | **BOI(タイ投資委員会)** |
| 期間 | **10年**(5年+5年延長) |
| 政府費用 | **50,000 THB/人**(10年分) |
| 扶養家族 | **最大4名**(配偶者+20歳未満の子4名) |
| Digital Work Permit | 任意・3,000 THB/年 |

### 5-2. 4カテゴリ詳細(2026年5月時点)

#### Category 1: Wealthy Global Citizens(富裕層)

| 要件 | 内容 |
|---|---|
| 年齢 | 制限なし |
| **資産** | **100万USD以上**(個人名義・グローバル資産) |
| タイ投資 | **50万USD以上**(国債/直接投資/不動産) |
| 年収 | **2025年2月に要件撤廃**(旧:80kUSD/年) |
| 健康保険 | 必須 |

#### Category 2: Wealthy Pensioners(リタイア層)

| 要件 | 内容 |
|---|---|
| 年齢 | **50歳以上** |
| パッシブ所得 | **80,000 USD/年**(年金・配当・賃料・利息等) |
| 代替条件 | 40,000 USD + タイ投資 250,000 USD |
| **対象外** | 給与所得・役員報酬は含まれない |

#### Category 3: Work-from-Thailand Professionals(リモートワーカー)

| 要件 | 内容 |
|---|---|
| 雇用先 | 海外の確立された大企業(上場 or 大規模私企業) |
| 所得 | **80,000 USD/年**(過去2年間) |
| 代替条件 | 40,000 USD + 修士号/IP/Series A資金調達 |
| タイでの就労 | 海外企業向けのリモートワーク前提 |

#### Category 4: Highly-Skilled Professionals(高度専門人材)

| 要件 | 内容 |
|---|---|
| 業界 | EV・バイオ・デジタル等(BOI指定の10産業) |
| 雇用先 | タイ企業・政府・教育機関 |
| 所得 | **80,000 USD/年** |
| 代替条件 | 40,000 USD + 修士号 |
| 学界の場合 | 一部例外あり |

### 5-3. LTR共通ベネフィット

```
✅ 10年マルチプルVISA(5年+5年延長)
✅ 外国所得への課税免除(WGC/WP/WFTP)
✅ 17%フラット税率(HSP のみ)
✅ 年次レポート(90日レポート不要)
✅ 4:1規則の免除(雇用比率規制免除)
✅ Digital Work Permit取得可能
✅ 空港ファストトラック
✅ 扶養家族最大4名
✅ 銀行口座開設可能
```

### 5-4. 申請プロセス(2段階)

```
[Step 1] BOI endorsement(無料・35,000 THB ※業者経由は変動)
       ↓
   承認後2ヶ月以内に
       ↓
[Step 2] VISA取得(50,000 THB)
   - One Bangkok(BOI窓口)で直接 or
   - タイ国外大使館で e-Visa経由
```

### 5-5. WALC対応方針

| 項目 | 内容 |
|---|---|
| 対応 | **専門スタッフ転送** |
| 強み | お抱え翻訳エージェントによる**公証翻訳が格安** |
| 実績 | 過去取得実績あり(櫻木さん等) |
| 標準対応 | 「専門的内容のため専門スタッフより回答」 |

## 6. Thailand Privilege(エリート / 2026年版)

### 6-1. 制度概要

| 項目 | 内容 |
|---|---|
| 旧名称 | Thailand Elite Visa |
| 2023年10月 | Thailand Privilege Visa に名称変更・5層構造に再編 |
| 運営 | Thailand Privilege Card Co., Ltd.(観光省傘下) |
| カテゴリ | 観光VISA(就労不可) |
| 銀行口座 | **開設可能** |
| 90日レポート | 代行サポートあり |

### 6-2. 5ティア詳細(2026年5月時点)

#### Bronze(エントリー)

| 項目 | 内容 |
|---|---|
| 価格 | **650,000 THB**(USD 19,000相当) |
| 期間 | **5年** |
| ポイント | **なし** |
| 家族追加 | **不可** |
| ステータス | ⏰ **2026年9月30日まで延長**(期間限定) |
| 主特典 | VIP空港・90日レポート代行・銀行口座開設サポート |

#### Gold

| 項目 | 内容 |
|---|---|
| 価格 | **900,000 THB**(USD 26,000相当) |
| 期間 | **5年** |
| ポイント | **20pt/年** |
| 家族追加 | 不可 |
| ステータス | 常設 |

#### Platinum

| 項目 | 内容 |
|---|---|
| 価格 | **1,500,000 THB**(USD 46,830相当) |
| 期間 | **10年** |
| ポイント | **35pt/年** |
| 家族追加 | 1,000,000 THB/人(通常) |

#### Diamond

|---|---|

#### Reserve

| 項目 | 内容 |
|---|---|
| 価格 | **5,000,000 THB**(USD 140,000相当) |
| 期間 | **20年** |
| ポイント | **120pt/年** |
| 家族追加 | 2,000,000 THB/人(通常) |
| ステータス | **招待制のみ**(年間100名限定) |

### 6-3. プロモーション(要随時確認)

| プロモ | 内容 | 期限 | 状況 |
|---|---|---|---|
| 申請料 50,000 THB免除 | 全ティア | 2025年12月31日(延長可能性) | 要最新確認 |
| Family Add 500,000 THB固定 | Platinum以上 | **2026年3月31日終了済** | 延長可能性あり |
| Bronze 延長 | 期間限定提供 | **2026年9月30日まで** | 確定 |

### 6-4. ポイントシステム

| ポイント数 | 利用例 |
|---|---|
| 1pt | 空港送迎・ラウンジ・映画・スパ・ホテル宿泊 |
| 2pt | 高級車送迎・健康診断・ゴルフ・VIPスポーツ |
| 3pt+ | 国内航空券・プライベートツアー・国際イベント |

### 6-5. WALC対応方針

| 項目 | 内容 |
|---|---|
| 対応 | **正規代理店として対応可能** |
| 強み | 取次・サポート |
| 推奨層 | 富裕層・面倒回避優先・銀行口座開設希望 |

## 7. SMART VISA(参考)

| 項目 | 内容 |
|---|---|
| 制度開始 | 2018年 |
| 対象分野 | EV・スマートエレ・医療・農業・ロボティクス・デジタル等 |
| 難易度 | **激高** |
